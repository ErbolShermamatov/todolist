# Todo List

Простой, быстрый и удобный менеджер задач с поддержкой темной темы и сохранением данных.
Приложение помогает организовать день.

##
**[Открыть сайт](https://erbolshermamatov.github.io/todolist/)**

## Функционал

* **Управление задачами:** Добавление, удаление и отметка "выполнено".
* **LocalStorage:** Задачи не исчезают после перезагрузки страницы. Приложение "помнит" ваш список.
* **Темная тема (Dark Mode):** Переключатель темы с сохранением выбора пользователя (если выбрали ночь, при следующем входе останется ночь).
* **Адаптивность:** Удобно пользоваться как на ПК, так и на телефоне.
* **Очистка:** Умное удаление элементов из DOM и памяти браузера.
